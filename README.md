# 使用

文章放在`_posts`目錄下，命名为`yyyy-MM-dd-xxxx-xxxx.md`，格式如下

```yaml
---
layout: post
title: 標題
categories: [分类1, 分类2]
---
文章内容，Markdown格式
```

文章資源放在`posts`目錄，文件名是`2019-05-01-theme-usage.md`，则该篇文章資源需要放在`posts/2019/05/01`下
在使用时直接引用即可。当然了，写作的时候会提示不存在忽略即可。
